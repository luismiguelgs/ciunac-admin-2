# Modulo Solicitudes - Tests

## Unitarias

- Parseo y normalizacion de IDs de solicitudes de beca

## Integracion

- Editar solicitud de constancias
- Rechazar solicitud
- Importar pagos CSV
- Crear solicitud de beca

## E2E

- Usuario autorizado gestiona solicitudes por estado
- Usuario autorizado importa pagos
- Usuario sin permiso es redirigido

## Casos negativos

- CSV de pagos invalido
- Rechazo de backend al actualizar solicitud
- Solicitud de beca inexistente
