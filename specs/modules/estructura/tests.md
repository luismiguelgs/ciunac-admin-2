# Modulo Estructura - Tests

## Unitarias

- Reglas de serializacion de opciones
- Helpers de ordenamiento o transformacion si se agregan

## Integracion

- Crear y editar idioma
- Crear y editar modulo con fechas validas
- Crear ciclo con idioma y nivel
- Editar texto por codigo

## E2E

- Usuario con `gestionar_estructura` entra al modulo y actualiza catalogos
- Usuario sin permiso es redirigido

## Casos negativos

- Intentar guardar modulo sin nombre o fechas inconsistentes
- Intentar crear ciclo sin idioma o nivel
- Error de red al cargar opciones
